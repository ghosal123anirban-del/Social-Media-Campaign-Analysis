### Digital Marketing Internship Project – SkillInfyTech IT Solutions

## 🔍 Project Overview

An interactive **Social Media Campaign Analysis Dashboard** built for **FreshBrew Coffee Co.** as part of the Digital Marketing Internship at SkillInfyTech IT Solutions.

This dashboard tracks and analyzes the performance of a 4-week social media campaign (April 2026) across **Instagram** and **LinkedIn**, covering reach, engagement, clicks, organic vs paid traffic, and KPI measurement.

> Open `Campaign_Interactive_Dashboard.html` in any browser to run the dashboard locally — no installation required.

## ✨ Features

- **Interactive Slicers** — Filter data by Platform, Week, and Traffic type in real time
- **KPI Cards** — Live updating metrics: Reach, Impressions, Engagement, Clicks
- **4 Dynamic Charts** — All charts update instantly based on selected filters
  - Reach over time (Bar chart)
  - Engagement over time (Line chart)
  - Organic vs Paid reach (Grouped bar chart)
  - Content type breakdown (Grouped bar chart)
- **Weekly Performance Table** — Filterable row-level data
- **Top Performing Posts Table** — Filtered by platform slicer
- **Reset Filter Button** — One-click reset to full view
- **Fully Offline** — Works without internet after download

## 🚀 How to Run

1. Download or clone this repository
2. Open `Campaign_Interactive_Dashboard.html` in any browser (Chrome / Edge / Firefox)
3. Use the **slicers** at the top to filter by Platform, Week, or Traffic type
4. Click **Reset filters** to return to the full view

No server, no installation, no dependencies needed.
